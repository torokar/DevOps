# Режимы работы Vim 

1) Режим команд (Command mode) - используется для выполнения команд. При запуске Vim, 
вы находитесь  в этом режиме
2) Режим вставки (Insert mode) - используется для ввода текста. Для перехода в этот режим,
нажмите клавишу 'i'
3) Режим замены (Replace mode) - используется для замены существующего текста. Для перехода
в этот режим, нажмите клавишу 'R'
4) Режим визуального выделения (Visual mode) - используется для выделения текста для 
копирования, вырезания или изменения. Для перехода в этот режим, нажмите клавишу 'v'



# Навигация 

1) h -переместить курсор влево
2) j - переместить курсор вниз 
3) k - переместить курсор вверх
4) l - переместить курсор вправо
5) w - переместить курсор на начало предыдущего слова
6) b - переместить курсор на начало предыдущего слова
7) е - переместить курсор на конец текущего слова
8) O - переместить курсор в начало строки
9) $ - переместить курсор в конец строки
10) gg - переместить курсоро в начало файла 
11) G - переместит курсор в конец файла




# Редактирование текста

1) i - вставить текста перед курсором
2) а - вставить текст после курсора
3) o - вставить новую строку после текущей строки и перейти в режим вставки 
4) dd - вырезать текущую строку 
5) yy - скопировать текущую строку 
6) р - вставить скопированный или вырезанные текст после курсора
7) u - отменить последнее действие 
8) Ctrl + r - повторит отмененное действие



# Сохранение и выход из редактора

1) :w - сохранить файл
2) :q - выйти из Vim
3) :wq - сохранить файл и выйти
